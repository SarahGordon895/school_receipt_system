## Insert into Word

1. Open your Word document
2. **Insert → Pictures → Picture from File**
3. Select the PNG from `word-style/png/`
4. Resize from corners; set **Wrap Text → Top and Bottom**

SVG files are also included if you want to edit shapes in Word (Insert → Pictures → SVG).
